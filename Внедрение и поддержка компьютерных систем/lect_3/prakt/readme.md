### Black Box Testing (Тестирование черного ящика)
- Расчесывает волосы как задумано?
- Работает для всех типов волос (короткие, длинные, густые, тонкие)?

#### Automated Testing (Автоматизированное тестирование)
- Выдержит ли автоматическое расчесывание роботом-манипулятором?
- Защита от статического электричества?
- Устойчивость к циклам автоматической мойки?
- Целостность после падения с конвейера?
- Совместимость с системами автоматической упаковки?

#### Functional Testing (Функциональное тестирование)
- Основная функция: расчесывает без вырывания волос?
- Разделяет пряди?
- Распутывает узлы?
- Массажирует кожу головы?
- Укладывает волосы (если предусмотрено дизайном)?

#### Validation (Валидация / Проверка соответствия)
- При тряске в поезде не сломается и не потеряется?
- Зубья есть или только ручка? (для декоративных моделей)
- Длина, толщина и расстояние между зубьями?
- Гибкость и прочность зубьев (не гнутся, не ломаются)?
- Концы зубьев закруглены? (Глаз не выколют?)
- Материал нетоксичен? (Съесть, отравимся?)
- Пожевать (для детских моделей - безопасность)
- Непроводящий материал? (Заснуть в розетку!)
- Если намочить (в ванной) и высушить, не деформируется? (Влагостойкость)
- Предел прочности на излом (Сломать)
- Устойчивость к падению с высоты (Уронить)
- Прочность при ударе (Постучать по столу)
- Гибкость всей конструкции (Попытаться согнуть)
- Инструкция по использованию/уходу в порядке?

#### Verification (Верификация / Проверка на соответствие спецификациям)
- Проверка: Соответствует ли заявленным материалам (пластик, дерево, металл)?
- Соответствует ли дизайну и цвету из макета?
- Правильность упаковки и маркировки?

#### Usability Testing (Тестирование удобства использования)
- Удобно лежит в руке? (Эргономика ручки)
- Удобно и эффективно расчесывает?
- Легко чистится от волос?
- Удобно хранить и брать с собой?

#### Compatibility Testing (Тестирование совместимости)
- Совместима с разными типами волос (прямые, кудрявые, окрашенные)?
- Совместима с разными средствами для укладки (гель, лак, мусс)?
- Не повреждает кожу головы?
- Безопасна для использования детьми?
- Какой срок годности/службы?
- Отдельно тестируется прочность зубьев
- Отдельно тестируется прочность ручки/основы

#### Performance Testing (Тестирование производительности/нагрузки)
- Сколько циклов расчесывания выдержит до поломки/деформации?
- Сколько раз можно уронить/ударить без повреждений?
- Какой срок службы до износа зубьев?
- Отдельно тестируется износостойкость зубьев
- Отдельно тестируется износостойкость ручки
